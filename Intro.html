What Is A computer?

Computer is a electronic device that process data according to giving instructions.
It takes input as a raw data, after the process we can get meaningful information and provide output or store information.

Parts Of A Computer

  Motherboard:



The main circuit board in a computer that allows all components to communicate with each other.The quality of the motherboard can impact performance, with more advanced boards can support fast RAM and improved smoother operation.

  Central Processing Unit(CPU):



We call for it brain of the computer. We use Central process unit (CPU) for process the input data and get information as output.

Example for CPU Manufacturers-:

Intel-i7 9700k,i9-6600f
Ryzen-5600X, 7700X
Apple-M1,A14 Bionic

  Random Access Memory(RAM):



RAM store data temporarily for quick access while the computer is running.The main types of RAM are Dynamic RAM (DRAM) and Static RAM (SRAM). DRAM is less expensive and used for main system memory. SRAM is faster than DRAM, more expensive, and used for CPU cache.

  Storage Devices:



These are use for store data permanently. We use HDD, SSD and NVME as our storage devices.
