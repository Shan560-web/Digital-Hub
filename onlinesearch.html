Online searching

Simply online searching means using search engines on the internet to get information by entering keywords.

Search Engines

Search engines help users find information on the internet. The most popular one is Google. Others include Bing, Yahoo, DuckDuckGo, Yandex, Baidu, and Ecosia.

using keyword and filters

keyword

Keywords are short phrases that tell the computer what you’re looking for. Instead of typing full sentences, use the main words.

For example:

Searching “how to make a new Gmail account” → instead use “Create Gmail Account” for faster and more accurate results.

Filters

Filters help to navigate what are you exactly looking for.
When search something on Google you can filter by

  • Images
  • Videos
  • News
  • Maps
  • Tools

When you use online shopping website like Ebay, Aliexpress or Daraz you can filter by like

  • Price range
  • Color
  • Shipping Method
  • Type
  • Brand

Using quotation mark(" ")

When you use quotation marks in a Google search, the engine looks for the exact phrase.

Example: Search for “importance of digital literacy” — the results will match the exact phrase.

Website Credibility

Be careful — not all information online is true! Website credibility shows how trustworthy a site is.

Website domain name help to identify trusted websites

These are the trusted domain names

  • .edu- educational website
  • .gov- government websites
  • .org- organizational website

Be cautious with .com or .net sites — anyone can create them. For example, https://www.amaz0n-billing.com looks familiar but it's fake.

Don’t believe every information first time you see it.
Check the other 2 or 3 sources and verify the information are real.
