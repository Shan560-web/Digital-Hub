What is the operating system?

The operating system (OS) can control everything your computer. It help you to manage programs and control your hardware resources.

Type of operating systems

  • Windows
  • MacOS
  • Linux
  • Android
  • MsDOS
  • Ubuntu

Desktop and icons

What is the desktop

The graphical user interface (GUI) or display area that appears on our screen is called a desktop on a computer.

Desktop Icons

There are some default icons are present on our desktop provided by windows. Some of the common icons on the desktop are This PC, Control Panel, Recycle Bin, internet browser, Network. Icons are shortcuts to open programs.


Files and folders

A file is a collection of data, such as a document or program. Files are stored within a folder. File mean something you save in your computer like, Documents, images, Videos or Audios.

How to create a new folder in your desktop

  1. Go to your desktop area.
  2. Right click and click new → folder.
  3. Give a name for folder. Example(My First Folder) and hit Enter to save.

How to rename a file or folder

  1. Locate you need to rename file/folder.
  2. Right click and select rename. Give a new name for file/folder

How to delete file or folder

  1. Click the file/folder to select it.
  2. Right click and select delete.

Or select the file/folder and press the delete key on keyboard. Then deleted File/folder move to recycle bin. If you needed it you can restore.

Delete permanently→ select file/folder and press Shift key + Delete key on the keyboard


Let's talk about File Extensions

Files split in two parts: Fisrt one is file name and other one is file extension.

Some example for file extensions :

  • name.txt: Text file
  • name.pdf: Portable Document Format
  • name.mp3: MP3 audio file
  • name.exe: Executable file
  • name.ppt: PowerPoint presentation
  • name.docx: Word Document

Property of the file

The "properties of a file" refer to details like the file's name, type, size, and location. Other common properties include the dates of creation, last modification and last access.

How to read file property

Right click on file→ properties
